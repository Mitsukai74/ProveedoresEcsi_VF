### 🏗 **Descripción del Proyecto**
Este un proyecto realizado para la etapa productiva en el programa de tecnología en análisis y desarrollo de sofware del SENA. Es un sistema de gestión de proveedores como aplicación web, desarrollado en **Spring Boot**, que permite, presentar y visualizar proveedores de la empresa ECSI SAS. Además, se ha migrado la base de datos de **MySQL a IBM DB2** y se ha implementado la funcionalidad para visualizar documentos asociados a cada proveedor.
